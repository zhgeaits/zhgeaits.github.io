MyEclipse

**一些常用快捷键**

* Shift+Enter 在当前行的下一行插入空行(这时鼠标可以在当前行的任一位置,不一定是最后) 
* Ctrl + Shift + O: 引入imports语句 
* Ctrl + Shift + T: 打开Open Type查找类文件 
* Ctrl + Shift + F: 整形 
* Ctrl + D : 删除本行 
* Ctrl + / : 注释本行
* Alt + Shift + j : 添加DOC注释 
* Alt+↓ 当前行和下面一行交互位置(特别实用,可以省去先剪切,再粘贴了)
* Alt+↑ 当前行和上面一行交互位置(同上)
